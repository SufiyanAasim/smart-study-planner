## Getting help

**For bugs:** open a [Bug Report issue](https://github.com/SufiyanAasim/smart-study-planner/issues/new?template=bug.yml).

**For feature requests:** open a [Feature Request issue](https://github.com/SufiyanAasim/smart-study-planner/issues/new?template=feature.yml).

**For questions:** open a [Question issue](https://github.com/SufiyanAasim/smart-study-planner/issues/new?template=question.yml) or email **sufiyanaasim@outlook.com**.

**For security vulnerabilities:** see [SECURITY.md](SECURITY.md) — do not open a public issue.

## Response time

This is a personal open-source project maintained by one developer. Response times are best-effort, typically within a few days for bugs and a week for feature discussions.
