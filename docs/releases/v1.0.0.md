# Smart Study Planner v1.0.0 — Insight

**Release date:** 2026-06-18
**Codename:** Insight
**Type:** Pre-release — internal development milestone, not publicly distributed.

---

## Overview

Initial project scaffold established. This milestone defined the data model, database schema, and core authentication flow that all subsequent releases build upon.

---

## Added

- Project scaffold: `main.py`, `database.py`, `models.py`, `logic.py`, `utils.py`.
- SQLite database schema design for `users` and `tasks` tables.
- Authentication proof-of-concept — register and login flow (security hardening completed in v1.0.5).
- `Task` dataclass with serialization and deserialization support.
- Basic CLI loop for task entry and listing.
