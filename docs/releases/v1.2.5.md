# Smart Study Planner v1.2.5 — Apex

**Release date:** 2026-06-29
**Codename:** Apex
**Type:** Release 5 — latest stable.

---

## Overview

Apex introduces LAN connectivity and academic identity. Users can now declare their department, class, section, and batch year on their profile. On the same local network, they can host named study rooms that peers discover automatically — no server, no configuration, no accounts required on the network level.

---

## Added

- **Academic Profile.** Registration and Settings → Academic Profile card now collect Department, Class/Batch, Section, and Batch Year. Fields are optional at registration and editable at any time. Academic info is stored in the database, displayed in the sidebar under the user's email, and broadcast with study rooms.
- **LAN Study Rooms panel.** A second sub-tab inside Study Groups labeled **🌐 LAN Study Rooms**. The panel has a live-updating room list on the left and a room detail/hosting card on the right.
- **Host a Study Room.** One click opens a dialog to name a room. The app then broadcasts a UDP beacon (port 50505) every 3 seconds on the local network. Peers see the room appear within seconds. Hosting continues until the user clicks Stop.
- **Room discovery.** Passive UDP listener surfaces all active rooms on the LAN, showing room name, host, and academic context (department · class · section). Rooms that stop broadcasting are automatically removed after 10 seconds.
- **`src/lan.py` — LANManager.** Self-contained, dependency-free module. All network I/O runs in daemon threads. UI updates are posted to the main thread via `Tk.after()` — the GUI never blocks.

## Changed

- Study Groups tab now contains two sub-tabs: **My Study Groups** (unchanged) and **🌐 LAN Study Rooms** (new). Discovery pauses when the tab is hidden; stops on logout.
- Sidebar profile header now shows Department · Class · Section when set.
- `users` table extended with `department`, `class_name`, `section`, `batch_year` columns. Existing databases migrate automatically via `ALTER TABLE`.
